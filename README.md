# Drone Recon Demo (UE5 C++)

A modular drone simulation project developed in Unreal Engine 5 using C++.

## Demo Video
[https://youtube.com/...](https://youtu.be/IXruFWwvqBM.)

## Features
- Custom drone movement system
- FPV camera
- Recon scanning system
- Artillery strike system
- Anti-air turret system

## Tech
- Unreal Engine 5
- C++
- AirSim

## Overview
This project demonstrates drone control, combat mechanics, and real-time simulation systems.
